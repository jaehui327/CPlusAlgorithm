입력이 몇 개인지 주어지지 않은 경우에는 입력을 EOF(End-of-File)까지 받으면 된다.
1. C
```c
while(scanf("%d %d", &a, &b) == 2)
```
2. C++
```c++
while(cin >> a >> b)
```
3. Java
```java
while(sc.hasNextInt())
```
